## SysMLv2 

The Flexo SysMLv2 service is an implementation of the OMG SysMLv2 API Services Specification, built on the Flexo architecture.

To try out the beta, see [Flexo SysMLv2 docker compose setup](https://github.com/Open-MBEE/flexo-mms-sysmlv2/tree/develop/docker-compose){:target="_blank"}, a Postman Collection file or [Bruno collection](https://github.com/Open-MBEE/flexo-mms-sysmlv2/tree/develop/bruno){:target="_blank"} is available for examples.


## Flexo MMS

Flexo MMS provides a graph-native approach to storing and diff'ing models. Models in Flexo MMS are not limited to enterprise model development platforms, it can store anything expressed as RDF.

To setup Flexo MMS without SysMLv2, see [Flexo MMS deployment](https://github.com/Open-MBEE/flexo-mms-deployment){:target="_blank"}
